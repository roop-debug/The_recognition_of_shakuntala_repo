# The_recognition_of_shakuntala_repo

The recognition of Shakuntala is a 2D narrative RPG made for a a college IKS project on the topic "Abhijnana Shakuntalam" with the theme of storytelling.<br>
We chose to develop a digital RPG game to bring the iconic Kalidasa play to life with multi-act game mechanics, interactive dialogue system, original artwork and immersive ambience and allow the original work a breath of fresh air by adapting it to a modern medium.

## Features
-Original art work with hand-drawn pixel art sprites<br>
-Multi-act dialogue system with branching narrative sequences<br>
-Distinct playable characters to fit the narrative of the original work<br>
-Synchronized Audio dialogues which gives the feeling of watching a play<br>

## Built with
-GameMakerStudio 2<br>
-Game Maker Language (GML)<br>

## Based on
Abhijnana Shakuntalam by Kalidasa
